# QCP跨链交易
QOS支持跨链交易，执行跨链交易前请确保[genesis](../spec/genesis.md)中qcps正确配置联盟链 或 已执行[创建QSC](../spec/txs/createqsc.md)、
[发放QSC](../spec/txs/issueqsc.md)等操作，以确保QOS链上保存有联盟链公钥信息、有账户上有联盟币资产。
