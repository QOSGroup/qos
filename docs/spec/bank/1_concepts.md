# 概念

## 地址

采用ed25519加密，Bech32编码

QOS地址区分如下：

| HRP               | Definition                            |
|-------------------|---------------------------------------|
| qosacc            | QOS账户地址                            |
| qosaccpub         | QOS账户公钥                            |
| qoscons           | QOS验证人共识地址                       |
| qosconspub        | QOS验证人共识公钥                       |
| qosval            | QOS验证人地址                          |
| qosvalpub         | QOS验证人公钥                          |
