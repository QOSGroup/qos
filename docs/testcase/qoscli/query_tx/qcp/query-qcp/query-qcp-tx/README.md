# Description
>     query qcp out tx from chainID and sequence.

>     根据chainID和sequence查询QCP Out Tx。

# Example

```
$ basecli qcp tx [chainID] --seq [Seq]
```

# Usage
```
  qoscli query qcp tx [chainID] [flags]
```
