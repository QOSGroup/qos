# Description
>     Get the tally of a proposal vote. 

>     获得提案投票的计票结果。

# Example

`qoscli query tally <proposal-id>`

查询编号为1的提议上实时统计结果：
```bash
$ qoscli query tally 1 --indent
```

查询结果：
```bash
{
  "yes": "100",
  "abstain": "0",
  "no": "0",
  "no_with_veto": "0"
}
```

# Usage
```
  qoscli query tally [proposal-id] [flags]
```
