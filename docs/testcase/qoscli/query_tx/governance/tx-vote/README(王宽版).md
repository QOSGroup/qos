# test case of qoscli tx vote

> `qoscli tx vote` 提议投票

---

qoscli tx vote 业务完整性的用例测试库, 包含以下部分:

* 情景1:
  
    投票的提议编号不存在，提议编号传入错误。

* 情景2：

    投票的选项错误，不在指定范围：Yes,Abstain,No,NoWithVeto。

* 情景3：

    投票的提议编号存在，但是提议的状态不是voting。

* 情景4：

    投票的提议编号传入正确，且投票选项在指定范围中。

* 情景5：

    查询提议的投票信息：查询某一提议上某一账号的投票信息；查询某一提议上所有投票信息。
